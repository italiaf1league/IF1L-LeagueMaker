# IF1L LeagueMaker v1.0.0

Prima release pubblica di IF1L LeagueMaker.

Applicazione desktop per gestire una lega F1 offline, con estrazione piloti, classifica campionato, inserimento manuale, export grafici e chiusura stagione.

## Download

### macOS Apple Silicon

Per Mac con chip M1, M2, M3 o successivi:

- `IF1L LeagueMaker-1.0.0-arm64.dmg`

### macOS Intel

Per Mac Intel:

- `IF1L LeagueMaker-1.0.0.dmg`

### Windows 64 bit

Per PC Windows x64:

- `IF1L LeagueMaker Setup 1.0.0.exe`

## Funzionalita principali

- Gestione lega offline
- Estrazione piloti e scuderie
- Inserimento manuale dei piloti
- Classifica campionato con modifica rapida dei punti
- Export JPG classifica in formato social
- Chiusura campionato con grafica campione e PDF riepilogativo
- Salvataggio locale automatico dei dati

## Note importanti

- I dati vengono salvati solo in locale sul computer che usa l'app.
- Al primo avvio macOS o Windows potrebbe mostrare un avviso di sicurezza perche l'app non e ancora firmata digitalmente.

### Se usi macOS

Se compare un blocco di sicurezza:

1. fai clic destro sull'app
2. scegli `Apri`
3. conferma di nuovo `Apri`

In alternativa:

1. vai su `Impostazioni di Sistema`
2. apri `Privacy e Sicurezza`
3. premi `Apri comunque`

### Se usi Windows

Se compare `Windows protected your PC`:

1. premi `More info`
2. premi `Run anyway`

## Requisiti

- macOS Intel oppure Apple Silicon
- Windows 10 o Windows 11 64 bit
